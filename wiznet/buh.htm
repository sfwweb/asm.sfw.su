Двухпроцессорный сервер для монтажа в стойку STSS Flagman RD2212

Сервер STSS Flagman RD2212 построен на базе двух высокопроизводительных процессоров AMD® Opteron® с использованием технологии Six Core. Данный сервер обладает высокой вычислительной мощностью по доступной цене. Flagman RD2212 может с успехом применяться, как универсальный сервер в средней компании. Обладая высокими вычислительными возможностями и сбалансированностью подсистем, он может использоваться в качестве сервера 1С:Предприятие 8.x с числом пользователей до 40-50.

Надежность обеспечивается благодаря использованию серверных компонентов, буферизированной памяти с коррекцией ошибок (ECC Registered), высококачественного источника питания с возможностью установки отказоустойчивого блока питания с горячей заменой, жестких дисков с возможностью горячей замены, аппаратных RAID-контроллеров, эффективной системы охлаждения и встроенным средствам диагностики и мониторинга.

Сервер может быть укомплектован дисковой подсистемой Serial ATA для обеспечения максимальной емкости данных, либо дисковой подсистемой SAS для обеспечения надежности и высокой производительности на приложениях типа СУБД. Сервер имеет 8 дисковых отсеков для установки жестких дисков с горячей заменой.

Сервер оптимизирован для установки в 19" стойку. Высота сервера в стойке - 2U.

Типовые задачи:

  • Сервер баз данных (40-50 пользователей)
  • Почтовый (mail) сервер
  • Сервер 1С Предприятие 8.x
